# 📈 Predictive Analytics Using Historical Data

## 🚀 Project Overview
This project focuses on predictive analytics using historical sales data.  
Machine Learning and Time-Series Forecasting techniques were used to predict future sales trends and visualize business insights.

---

# 🔥 Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis
- Interactive Visualizations
- Linear Regression Forecasting
- ARIMA Time-Series Forecasting
- Future Sales Prediction
- Power BI Interactive Dashboard

---

# 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Plotly
- Matplotlib
- Scikit-learn
- Statsmodels
- Power BI

---

# 📊 Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Monthly Sales Trend
- Region Wise Analysis
- Category Sales Distribution
- Future Sales Forecast
- Interactive Filters & Slicers

---

# 📁 Files Included
- Predictive_Analytics.ipynb
- Sample - Superstore.csv
- future_sales_forecast.csv
- Predictive_Analytics_Dashboard.pbix

---

# 🎯 Project Outcome
This project demonstrates predictive modeling, forecasting techniques, data visualization, and business intelligence dashboard development using Python and Power BI.
